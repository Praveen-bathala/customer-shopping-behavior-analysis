# customer-shopping-behavior-analysis
End-to-end data analytics project analyzing customer shopping behavior using Python, SQL, and Power BI to uncover purchasing patterns, customer segments, and business insights.

Tools Used
Python (Pandas, NumPy, Matplotlib, Seaborn)
MySQL
Power BI
CSV Dataset
Dataset Information

The dataset contains:

Customer ID
Age
Gender
Category
Purchase Amount
Location
Season
Review Rating
Subscription Status
Discount Applied
Payment Method
Frequency of Purchases
Project Workflow
1. Data Cleaning (Python)
Handled missing values
Standardized column names
Created Age Groups
Performed exploratory data analysis
2. SQL Analysis

Business questions solved:

Revenue by Gender
Top Rated Products
Impact of Discounts on Spending
Shipping Type Analysis
Subscriber vs Non-Subscriber Spending
3. Power BI Dashboard

Dashboard includes:

Total Revenue
Average Purchase Amount
Customer Segmentation
Seasonal Sales Trends
Purchase Frequency Analysis
Gender-wise Revenue Distribution
Key Insights
Identified high-spending customer segments.
Analyzed purchasing trends across categories.
Evaluated impact of discounts and subscriptions.
Discovered seasonal shopping patterns.
